Règles d'inférences
  • Modus ponens : {A ⇒ B, A} ⊨ B (si A implique B, et A vrai : alors B vrai)
  • Modus tollens : {A ⇒ B, ¬B} ⊨ ¬A (si A implique B, et B faux : alors A faux)
  • Syllogisme : {A ⇒ B, A ⇒ B} ⊨ A ⇒ C (si A implique B, et B implique C : alors A implique C)
Schémas d'axiomes
  • SA1 : (A ⇒ (B ⇒ A))
  • SA2 : ((A ⇒ (B ⇒ C)) ⇒ ((A ⇒ B) ⇒ (A ⇒ C)))
  • SA3 : ((¬A ⇒ ¬B) ⇒ (B ⇒ A))

Univers de Herbrand

Définition : Soit C un enmselbe de clauses. Considérons H0 l'ensemble des symboles de constantes ayant au moins une occurence dans C.
On définit Hi ensemble des termes clos de C de niveau i par : Hi = Hi-1 U {f(t1, ..., tn)}
H∞ = limi⇒∞ Hi est appelé univers de Herbrand de C.
Remarque : si aucune constante n'apparaît dans C, on pose H0 = {a}.
Exemples :
Exemple 1 :
Soit C = {P(a), ¬P(x) ∨ P(f(x))}. On a :
H0 = {a}
H1 = H0 U {f(a)} = {a, f(a)}
...
Hi = {a, f(a), f2(a), ..., fi-1(a)} U {fi(a)}
...
H∞ = {a, fn(a) | n∈N*}
Exemple 2 :
Soit C = {P(f(x)), R(a, g(y), b)}. On a :
H0 = {a, b}
H1 = H0 U {f(a), f(b), g(a), g(b)}
H2 = H1 U {f2(a), f2(b), g2(a), g2(b), f(g(a)), f(g(b)), g(f(a)), g(f(b))}
...
H∞ = {fn1(gn2(fn3( ... (a) ...))), fm1(gm2(fm3( ... (b) ...))) | ∀nj,mj∈N, j∈N*}
Atome de Herbrand : On appelle atomes de Herbrand associés à un ensemble de clauses C, les atomes obtenus en remplaçant dans les atomes de C les variables par des éléments de H∞. Base de Herbrand : L'ensemble des atomes de Herbrand est appelé base de Herbrand. Réalisation de base : On appelle réalisation de base d'une clause C, une clause obtenue en^remplaçant les variables de C par des éléments de H∞
Exemple :
Soit C = {P(a, f(x)), Q(b) ∨ ¬R(g(y))}.
univers de Herbrand : H∞ = {a, b, f(a), f(b), g(a), g(b), f(g(a)), ...}
atomes de Herbrand : P(a, f(a)), P(a, f(g(b))), Q(b), R(g(a))
réalisation de base :
  • P(a, f(a)), Q(b) ∨ ¬R(g(a))
  • P(a, g(b)), Q(b) ∨ ¬R(g(g(b)))

Forme prénexe

= quantificateurs (∀/∃) apparaissent à gauche
Exemple de mise sous forme prénexe
∀yR(z, y) ⇒ ∀zS(z, y))            formule de base
∀y(¬R(z, y) ⇒ ∀zS(z, y)))       déplacement de ∀y (avec négation de R)
∀y(¬R(x, y) ⇒ ∀zS(z, y)))       renommage du 1er z en x
∀y∀z(¬R(x, y) ∨ S(z, y)))       déplacement de ∀z (avec modification du ⇒ en ∨)

FNCC (forme normale conjonctive canonique)

= conjonction (∧) d'une ou plusieurs disjonctions (∨) d'un ou plusieurs littéraux
Exemples
  • A ∧ B
  • A ∨ B
  • (A ∨ B) ∧ C
  • (A ∨ ¬B ∨ ¬C) ∧ (¬D ∨ E ∨ F)
Contre-exemples
  • ¬(A ∧ B)                     négation devant formule
  • A ∧ (B ∨ (C ∧ D))      ∧ imbriqué dans ∨

FNDC (forme normale disjonctive canonique)

= disjonction (∨) d'une ou plusieurs conjonctions (∧) d'un ou plusieurs littéraux
Exemples
  • A ∨ B
  • (A ∧ B) ∨ C
  • (A ∧ ¬B ∧ ¬C) ∨ (¬D ∧ E ∧ F)
Contre-exemples
  • ¬(A ∨ B)                     négation devant formule
  • A ∨ (B ∧ (C ∨ D))      ∨ imbriqué dans ∧
